# Velvet Vogue - E-commerce Website

##  Description

This is a PHP & MySQL based e-commerce website developed for a clothing store.

##  Features

* Product browsing
* Shopping cart
* User login/register
* Admin panel

##  Technologies

* PHP
* MySQL
* HTML, CSS, JavaScript

##  How to Run

1. Install XAMPP
2. Copy project to htdocs
3. Import database.sql into phpMyAdmin
4. Run Apache & MySQL
5. Open http://localhost/project-folder
